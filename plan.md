# Piano Self-Learning/Teaching App

## What does it do?
- Create an app that should be able to help users to learn piano step by step. whether if they're complete learners or experienced players or even people who used to play piano/keyboard long time ago but want to relearn it now should be able to be useful off this app.

## What does it focus on?
- Focus on all sorts of music of every genere and also languages. but for now let's keep Tamil, English and anime songs, albums, OSTs as the priority to focus.

## Features
- Easily navigatable
- should be able to add a song and get notes out of it so that we can easily play it out.
- track progress of each plays based off of the user's skill level.
- create sheetmusic for songs, OSTs when requested by the user (not important now but can be added later)
- basically it should be a personal, super helpful piano tutor 

## Application
- Use relevant tech stacks that seem fit (FE and BE and maybe something for DB if necessary)
- better if any enivronmental config is not used for now. but you can add if it is seemed absolutely necessary.
